## Reference Engine Patterns To Mine Carefully

The simulator includes ten deliberately separated reference engines. Do not
publish these as personality types. Use them privately to generate contrastive
story angles.

- Secure Integrator: flexible regulation and trust.
- Anxious Pleaser: approval-seeking under threat.
- Avoidant Self-Reliant: independence-protective withdrawal.
- Vigilant Controller: safety-driven control seeking.
- Grandiose Striver: status and mastery pressure.
- Depleted Withdrawer: low-energy avoidance after failure cues.
- Principled Idealist: fairness-sensitive value defense.
- Pragmatic Maximizer: analytical, utility-heavy choice style.
- Enmeshed Caretaker: other-directed over-functioning.
- Detached Intellectualizer: abstraction under ambiguity.

## Public Claim Safety

Use these phrases:

- "One way to understand this pattern is..."
- "In therapy-adjacent language, this can look like..."
- "This is not a diagnosis; it is a lens."
- "The useful question is not 'what is wrong with me?' but 'what did this
  situation activate?'"

Avoid these phrases:

- "This means you have..."
- "Your schema is..."
- "People like this always..."
- "The simulator predicts..."
- "Science proves..."
