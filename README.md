# PsychSimulator Outreach

This repo is the planning and tracking home for an outreach blog series based on
the PsychSimulator and Psych-LLM-Wiki projects.

The content goal is to build Lacey's public voice around relatable personal and
professional stories that make therapy-adjacent concepts useful in daily life.
Posts should be approachable, funny where natural, evidence-aware, and careful
not to overclaim clinical certainty. The simulator and knowledge graph provide
the concept map and provenance spine, but posts do not need to mention either
project unless it helps the story.

## Workflow

1. Open a GitHub issue for each post idea using the blog post issue template.
2. Build the outline from the content roadmap and provenance map.
3. Draft in this repo or in Lacey's GitHub Pages repo, depending on who is
   editing.
4. Lacey reviews voice, clinical framing, and academic caution.
5. Publish on GitHub Pages.
6. Create LinkedIn, Instagram, and Facebook variants from the same source post.
